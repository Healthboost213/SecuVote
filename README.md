# SecuVote

SecuVote is a secure voting platform built with Flask. Built with security, resiliency and reliability in mind.
It is meant to be integrated cleanly for organisations relying on Google Workspace.

### Features:
🎨 Easy-to-use responsive UI design  
💪 Robust Configuration of Postings and Candidates  
🔐 Google OAuth Login using Flask-Dance  
💉 Preventing SQLi Attacks using SQLAlchemy  
👤 Total User Anonymity and Vote Hashing
